# 🎮 D&D Console RPG (Java)

Консольная RPG-игра, разработанная на Java в стиле Dungeons & Dragons.  
Проект создан для практики объектно-ориентированного программирования (ООП), архитектуры приложений и работы с Java Core.

---

## 📖 Описание

Игрок управляет персонажем и сражается с противниками в пошаговых боях.  
В основе проекта лежит механика D&D: броски кубиков, характеристики персонажей и система урона.

Проект разработан как pet-project для углубленного изучения Java.

---

## ⚙️ Функционал

- Создание персонажа
- Система пошаговых боев
- Генерация врагов
- Механика бросков кубиков (d20)
- Расчет урона
- Классы персонажей и врагов (ООП)
- Наследование и полиморфизм
- Работа с коллекциями Java
- Обработка исключений

---

## 🛠️ Используемые технологии

- Java (Core)
- ООП (Inheritance, Polymorphism, Encapsulation)
- Collections Framework
- Exception Handling
- Git & GitHub
- IntelliJ IDEA

Вариант 1 (самый лучший) — добавить проект в раздел «Опыт работы»

Компания: Pet Project (или Personal Project)

Должность: Junior Java Developer

Период: январь 2026 — настоящее время

Описание:

    Разрабатываю консольную RPG по мотивам Dungeons & Dragons на Java.

    В рамках проекта реализованы:

        броски кубиков;

        работа с коллекциями Java;

        обработка исключений;

        взаимодействие между классами;

        подключение PostgreSQL (если уже есть).

Это абсолютно нормальная практика. Многие джуниоры именно так и оформляют pet-проекты.
